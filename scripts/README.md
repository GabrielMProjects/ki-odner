# scripts

Hilfsskripte für Setup, lokalen Betrieb und Aufräumen.

Geplant:
- `setup-local.sh` – lokale Entwicklungsumgebung hochziehen.
- `build.sh` – Frontend/Backend bauen.
- **`destroy.sh`** – **entfernt nach der Demo alle erstellten AWS-Ressourcen** wieder
  (Pflicht, um Kosten zu vermeiden).

> Die Skripte werden in späteren Abschnitten befüllt.

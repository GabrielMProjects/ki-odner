# helm/angel-lara

Helm-Chart für das Kubernetes-Deployment von **angel** (Frontend) und **lara** (Backend).

**Zweck:** Optionale, fortgeschrittene Deploymentvariante (containerisiert, K8s).
Lokal testbar mit **minikube** oder **kind**, bevor ein Cloud-Cluster (z. B. EKS) genutzt wird.

> `Chart.yaml`, `values.yaml` und Templates werden in einem späteren Abschnitt befüllt.
